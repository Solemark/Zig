# Zig Practice

create a new zig project with:

    zig init-exe
    OR
    zig init-lib

run a zig file with:

    zig run <path_to_main_file>.zig

run a test with:

    zig test <path_to_test_file>.zig

NOTE:
pub vs export:

    pub = visible to files inside the module
    export = visible outside of the module
